Ringkasan

Tools ini berjalan di browser menggunakan ffmpeg.wasm (untuk versi offline) atau bisa dipasangkan ke server jika Anda ingin proses YouTube otomatis. Ikuti petunjuk pada halaman tools.

1

Split top & bottom

Mengambil dua crop (atas & bawah) dari satu video dan menggabungkannya menjadi vertikal 9:16.

2

Output 1080×1920 (default)

Kualitas tinggi, cocok untuk short reels & story.

3

Tanpa server (opsional)

Bisa berjalan offline di browser (ffmpeg.wasm) atau melalui Termux + ffmpeg untuk hasil stabil di HP.

Cara pakai singkat

  1. Pilih Tools → pilih file video lokal (atau unduh YouTube terlebih dahulu).
  2. Klik Proses jadi 9:16 dan tunggu sampai selesai.
  3. Unduh hasil; untuk file besar gunakan Termux + native ffmpeg jika ffmpeg.wasm gagal.
